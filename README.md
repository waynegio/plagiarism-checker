# Plagiarism Checker
A web-based plagiarism checker built with Python and Streamlit using the MSR Paraphrase Corpus dataset. This project analyzes whether two sentences are paraphrases (have similar meaning) or not.

To run this project locally, install the required dependencies using `pip install -r requirements.txt` and then start the app with `streamlit run Home.py`. Open the link shown in the terminal (usually http://localhost:8501) in your browser.

You can also access the deployed version globally via: https://paircheck.streamlit.app/
